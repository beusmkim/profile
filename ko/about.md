# About

저는 **하드웨어 특성을 고려한 AI 소프트웨어/시스템**을 지향합니다.  
정확도뿐 아니라 **메모리·대역폭·실행 특성**이 실제 성능을 결정한다고 믿습니다.

## 제가 하는 일
AI 모델과 하드웨어 실행 사이에서 아래를 설계/최적화합니다.

- CUDA 인지형 추론 설계(스트림/비동기 실행 개념)  
- FP16/INT8 정밀도 전략 및 품질-성능 트레이드오프  
- VRAM 풋프린트/할당 패턴/대역폭 병목 관점 최적화  
- 동시성에서 안정적인 처리량을 위한 compute isolation 및 워커 스케일링  

## 실리콘(AI 칩)과의 연결
AI 칩은 소프트웨어가 아래를 해낼 때 성능이 납니다.
- 메모리 스톨을 줄이며 가속기를 지속적으로 “먹여주기”  
- 불필요한 전송/중복 그래프를 제거해 실행 효율 높이기  
- 실제 모델 동작 기반으로 성능을 검증/개선하기  

## 정량 성과
- FP16 적용으로 **~38–40%** peak VRAM 감소  
- **~22–27%** 추론 지연 개선  
- 동시 요청 환경에서 **~35%** blocking 감소
