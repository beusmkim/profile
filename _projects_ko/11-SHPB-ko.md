## Overview

본 프로젝트는 Split Hopkinson Pressure Bar(SHPB) 실험을 통해 고변형률 환경에서의 재료 거동을 계측하고, 동적 하중 조건에 맞는 구성방정식을 보정하는 것을 목표로 했다.

실험 데이터를 기반으로 예측 가능한 수치 모델을 구축하는 것이 핵심이었다.

---

## Problem

재료는 고변형률 조건에서 정적 조건과 다른 거동을 보인다.

주요 문제:

- 실험 stress–strain 곡선에서 변형률 속도 의존성 확인
- 기존 구성방정식 파라미터로는 동적 거동 예측 불가
- 유한요소 해석 결과가 실험값과 차이 발생

실험 기반 모델 보정이 필요했다.

---

## Experimental Setup

- Split Hopkinson Pressure Bar(SHPB) 장치 사용
- 고속 충격 하중 실험
- 동적 응력–변형률 곡선 추출
- 파형 데이터 신호 처리 및 분석

---

## Diagnosis

분석 결과:

- 재료 거동에서 변형률 속도 민감도 확인
- 충격 조건에서 비선형 경화 특성 관찰
- 기존 SK 모델 파라미터는 고변형률 영역을 충분히 반영하지 못함

실험–해석 간 오차는 파라미터 불일치에서 기인했다.

---

## Approach

### 구성방정식 보정

- SK 모델 적용
- 실험 데이터에 맞춰 변형률 속도 의존 파라미터 피팅
- 반복적 계수 조정을 통해 오차 최소화

### 유한요소 검증

- 보정된 모델을 Abaqus에 적용
- SHPB 조건과 동일한 충격 시뮬레이션 수행
- 실험 stress–strain 곡선과 비교

---

## Result

- 실험 대비 시뮬레이션 오차 감소
- 동적 하중 조건에서 예측 정확도 향상
- 고변형률 조건에 대한 검증된 파라미터 세트 확보

---

## Limitations

- 검증은 실험 범위 내 변형률 속도에 한정
- 등방성 가정 기반 모델
- 극한 조건에서는 추가 미세구조 모델 필요

---

## Key Insight

극한 조건에서의 시스템 거동을 정확히 예측하려면 실험 데이터 기반의 보정과 반복 검증이 필수적이다.

이 과정은 하드웨어 성능 검증에서 측정 → 모델링 → 보정 → 검증으로 이어지는 구조와 유사하다.
